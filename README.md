# Modbus Client GUI

Графический клиент для работы с протоколом Modbus (аналог Modbus Poll). Поддерживает подключение по **Modbus TCP** и **Modbus RTU** (через последовательный порт).

## Возможности

- Подключение по **Modbus TCP** (IP-адрес, порт)
- Подключение по **Modbus RTU** (последовательный порт, скорость передачи)
- Чтение данных через основные функции Modbus:
  - **FC01** — Read Coils
  - **FC02** — Read Discrete Inputs
  - **FC03** — Read Holding Registers
  - **FC04** — Read Input Registers
- Автоматический опрос данных (polling) с настраиваемым интервалом
- Отображение результатов в таблице с форматами: десятичное, двоичное, шестнадцатеричное
- Индикация статуса подключения и ошибок

## Требования

- Python 3.8+
- PyQt6
- pymodbus

## Установка зависимостей

```bash
pip install PyQt6 pymodbus
```

## Запуск

```bash
python modbus_client.py
```

## Использование

1. Выберите тип подключения: **Modbus TCP** или **Modbus RTU**.
2. Укажите параметры подключения:
   - Для TCP: IP-адрес хоста и порт (по умолчанию `127.0.0.1:502`).
   - Для RTU: последовательный порт и скорость передачи (по умолчанию `/dev/ttyUSB0`, 9600 бод).
3. Укажите **Slave ID** устройства (по умолчанию `1`).
4. Нажмите **«Подключиться»**.
5. Выберите функцию чтения (FC01–FC04) или настройте автоматический опрос.
6. Укажите начальный адрес (**Start Address**) и количество данных (**Quantity**).
7. Результаты отобразятся в таблице с адресами, значениями и представлениями в двоичном и шестнадцатеричном форматах.

## Структура интерфейса

- **Настройки подключения** — выбор типа подключения и параметров.
- **Кнопки управления** — подключение/отключение и выбор функции чтения.
- **Параметры запроса** — адрес начала чтения и количество регистров/бит.
- **Таблица результатов** — отображение полученных данных.
- **Статус бар** — текущее состояние подключения и результаты операций.

## Примеры

### Подключение к TCP-устройству

1. Выберите **Modbus TCP**.
2. Введите IP-адрес устройства (например, `192.168.1.100`).
3. Оставьте порт `502` (или укажите свой).
4. Нажмите **«Подключиться»**.

### Подключение к RTU-устройству

1. Выберите **Modbus RTU (Serial)**.
2. Укажите порт (например, `COM1` или `/dev/ttyUSB0`).
3. Установите скорость передачи (например, `9600`).
4. Нажмите **«Подключиться»**.
